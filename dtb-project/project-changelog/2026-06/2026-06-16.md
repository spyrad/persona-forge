### Befunde / Gotchas

- **`astro check` ≠ build:** `npm run build` (esbuild) und `npm run lint`
  type-checken nicht voll; `npx astro check` (kein npm-Script) fand echte
  tsc-Fehler. Künftig vor Phasen-Commit mitlaufen lassen.
- **Supabase-Client untypisiert (kein `Database`-Generic):** Query-`data` ist mal
  `any` (custom Select → `no-unsafe-assignment`, expliziter Row-Cast nötig), mal
  schon getypt (`no-unnecessary-type-assertion`). Inkonsistent → fallweise per
  Lint entscheiden.
- **`db push` non-interaktiv:** mit `< /dev/null` läuft der `[Y/n]`-Prompt auf
  Default (Yes) durch; Credentials kommen aus dem Keyring (kein Passwort-Prompt).
- **React-Types:** `React.FormEvent` ist deprecated → Codebase nutzt
  `React.SubmitEvent<HTMLFormElement>`.
